# 🗺️ Transit & Tour Master API

Интеллектуальный оркестратор городских маршрутов. Объединяет нейросетевой чат, геосервисы 2ГИС/Яндекс и облачную БД MongoDB.

## 🚀 Архитектура и Схема взаимодействия

Проект работает по принципу **"Стейт-машины через БД"**. Сервер выступает посредником между Нейросетью и Фронтендом.

### Потоки данных:
1. **Диалог:** `Frontend <-> API <-> Нейросеть` (Через роут `/api/chat`).
2. **Инициация:** `Нейросеть -> API` (Роут `/api/search_places`). Сервер ищет реальные объекты и складывает их в коллекцию `suggestions`.
3. **Синхронизация:** `Frontend -> API` (Роут `/api/get_suggestions`). Фронтенд забирает найденные варианты.
4. **Сборка:** `Frontend -> API` (Роут `/api/generate_route`). Сервер решает задачу коммивояжера и ищет расписания транспорта.
5. **Финал:** `Frontend -> API` (Роут `/api/get_user_route`). Получение готового маршрута из коллекции `generated_tours`.

## 🛠 Эндпоинты

### Для Нейросети (Webhook)
* **POST `/api/search_places`**: Принимает теги (категории, старт). Запускает фоновый поиск в GIS-сервисах и сохраняет результат в MongoDB (буфер предложений).

### Для Frontend-разработчика

#### 1. Чат (Streaming)
* **POST `/api/chat`**: Пересылает сообщения нейросети. Возвращает `text/plain` поток.

#### 2. Получение вариантов (Polling)
* **GET `/api/get_suggestions/{login}`**: Проверка готовности списка мест.
  * `404`: Места еще ищутся.
  * `200`: Список объектов готов для отображения пользователю.

#### 3. Генерация маршрута
* **POST `/api/generate_route`**: Отправка выбранных пользователем карточек. Запускает оптимизацию пути и сбор расписаний.
* **Body:** содержит `login`, `transport_mode`, `start_point` и массив `selected_places`.

#### 4. Получение финала (Polling)
* **GET `/api/get_user_route/{login}`**: Возвращает полностью готовый маршрут с отсортированными шагами и расписанием транспорта.

## 📦 Инфраструктура
* **FastAPI**: Основной сервер.
* **MongoDB Atlas**: Облачное хранилище (Коллекции `suggestions` и `generated_tours`).
* **Background Tasks**: Все тяжелые вычисления (GIS, TSP) вынесены в фон для мгновенного ответа API.
